## **Privacy policy**
This policy explains how the server processes data. The Bedrock dedicated server temporarily log usernames, XUIDs, and PFIDs when players join or disconnect.

1. Logs including username and datetime are posted in the Discord channel "player-connected."
2. Only server administrators can view full logs (including XUID and PFID);
3. regular members cannot see XUID or PFID.

An option to hide join/disconnect notifications is planned. For questions, contact the server administrators.
